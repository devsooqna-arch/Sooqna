# إجراءاتك المؤجلة (تنفيذ دفعة واحدة)

هذا الفولدر يجمع فقط الأشياء المطلوبة منك، بحيث لا تنفذها الآن.
نفّذها مرة واحدة عندما ننتهي من كل التعديلات.

## 1) تجهيز متغيرات البيئة النهائية

- backend: افتح `backend.todo.env` واملأ القيم الناقصة.
- web: افتح `web.todo.env` واملأ القيم الناقصة.

## 2) تفعيل سياسات Week 2 (Auth Hardening)

- في backend:
  - `REQUIRE_EMAIL_VERIFIED=true`
  - `RECAPTCHA_ENABLED=true`
  - `RECAPTCHA_SECRET_KEY=<your-secret>`
- في web:
  - `NEXT_PUBLIC_RECAPTCHA_SITE_KEY=<your-site-key>`

## 3) تنفيذ تحقق نهائي سريع

- تشغيل الباك والويب.
- اختبار تسجيل جديد:
  - لا يسمح بالاستمرار بدون توثيق البريد.
  - زر "إعادة إرسال رابط التحقق" يعمل.
- اختبار تسجيل الدخول:
  - reCAPTCHA يمر بنجاح.
- اختبار endpoint محمي (مثل إنشاء إعلان):
  - مستخدم غير موثق: مرفوض.
  - مستخدم موثق: مسموح.

## 4) قاعدة تنفيذ

- لا تخلط بيئات local/production.
- نفّذ هذه القائمة مرة واحدة فقط بعد انتهاء كل المهام المطلوبة في الأسبوع.
