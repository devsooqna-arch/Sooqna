# Sooqna

Monorepo layout:

- **apps/web** — web client
- **apps/mobile** — mobile client
- **backend/functions** — serverless functions (e.g. Firebase Cloud Functions)
- **backend/shared** — shared backend code
- **backend/config** — deployment and environment configuration
- **docs** — project documentation

Firebase configuration at the repository root: `firestore.rules`, `firestore.indexes.json`, `storage.rules`.
